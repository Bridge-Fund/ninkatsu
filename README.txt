【すぐに使う方法（Googleカレンダー連携なし）】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. スマホに4ファイルをコピーする（LINEで送るか、USB接続）
2. スマホのChromeで index.html を開く
3. Chromeのメニュー（⋮）→「ホーム画面に追加」
4. ホーム画面にアプリアイコンが追加されます！

これだけで、オフラインでも動くアプリとして使えます。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【Googleカレンダーと連携する方法】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

連携すると、排卵日・生理予定・服薬スケジュールが
Googleカレンダーの「妊活ノート」カレンダーに自動登録されます。

■ 手順

STEP 1: Google Cloud Console でプロジェクト作成
  → https://console.cloud.google.com
  → 「新しいプロジェクト」→ 名前: ninkatu-note

STEP 2: Google Calendar API を有効化
  → 「APIとサービス」→「ライブラリ」
  → 「Google Calendar API」を検索 → 「有効にする」

STEP 3: OAuth 同意画面を設定
  → 「APIとサービス」→「OAuth同意画面」
  → ユーザーの種類: 外部
  → アプリ名: 妊活ノート
  → スコープ: .../auth/calendar を追加
  → テストユーザー: 奥様のGmailアドレスを追加

STEP 4: OAuth クライアントID を作成
  → 「APIとサービス」→「認証情報」
  → 「認証情報を作成」→「OAuthクライアントID」
  → アプリケーションの種類: 「ウェブアプリケーション」
  → 承認済みのJavaScript生成元に追加:
      file://   （ローカルファイルから開く場合）
      http://localhost （ローカルサーバーの場合）
  → 作成後、「クライアントID」をコピー

STEP 5: app.js にクライアントIDを設定
  → app.js をテキストエディタで開く
  → 以下の行を探す:
      gcalClientId: '',
  → クライアントIDを貼り付ける:
      gcalClientId: '123456789-xxxx.apps.googleusercontent.com',

STEP 6: アプリで「📅」ボタン →「Googleアカウントで連携する」をタップ

【アプリの使い方】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ ホーム画面
  ・排卵日・生理日までのカウントダウンを確認
  ・「生理開始日を記録」で生理記録 → 予測が自動更新
  ・今日の服薬・注射をタップでチェック ✓

■ カレンダー画面
  ・すべての情報を1つのカレンダーで確認
  ・色の意味:
    🔴 ピンク = 生理期間
    ⭐ 紫　　= 排卵予測日
    💜 薄紫　= タイミング推奨日
    💊 紫ドット = 服薬
    💉 青ドット = 注射
    📅 緑　　= Googleカレンダーの個人予定

■ 予測画面
  ・次回排卵日・生理予定日を確認
  ・タイミング推奨日（排卵2日前〜1日後）

■ 先生画面
  ・担当医ごとに診察曜日・時間帯を登録

■ メモ画面
  ・毎日の気分（絵文字）・基礎体温・フリーテキストを記録
  ・◀▶で日付を変えて過去のメモも確認できます

■ ホーム画面 右上アイコン
  📅 = Googleカレンダー連携
  💊 = 服薬・注射の管理（追加・ON/OFF）

【データについて】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

・データはスマホ本体（ブラウザのlocalStorage）に保存されます
・ブラウザのデータを消去するとデータが消えます
・大事なデータはGoogleカレンダーに同期しておくと安心です
・機種変更時は新しい端末でも同じファイルを開けば使えます
